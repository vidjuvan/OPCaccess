OPCaccess Class

Revision: 1.3 - Author: meyer






Device Properties
Property name
Property type
Description
DataFromCache Tango::DEV_SHORT Triggers data reading from the OPC cache or directly from the OPC item. Set to 1 -> cache reading is enabled. Set to 0 -> direct access is enabled.
Server_name Tango::DEV_STRING Specifies the name of the OPC server to be used.
Path_name Tango::DEV_STRING Specifies the path of the OPC item to be addressed on the OPC server.
Item_names Array of string Specifies the OPC item names to be read from the OPC server via the given path.
Data_type Tango::DEV_STRING This is set as the requested data type for the OPC items to be read. The possible data types are "double", "long" or "short".
Array_handling Tango::DEV_STRING Specifies whether the OPC server used is capable to handle arrays or only scalar data types. array_handling = "true" , array data types are handled. array_handling = "false", only scalar data types are handled.



There is no Class properties.
